## 💡 Important Notes

### Headless Browser Automation (2026-02-11)

**Challenge:** Servers without display (X11) cannot run standard browsers

**Solutions for Playwright/Selenium:**

1. **Cookie Export Method (BEST):**
   - Export cookies from local browser via EditThisCookie extension
   - Import to Playwright storage state
   - No display needed, works 100% headless
   - Use for: LinkedIn, Twitter, any authenticated site

2. **Xvfb Virtual Display:**
   ```bash
   xvfb-run :99 -s '-screen 0 1280x720x24' command
   ```
   - Provides virtual X11 display
   - Allows interactive setup
   - Use for: Initial login, complex workflows

3. **Root Sandbox Workaround:**
   ```python
   launch_options = {
       "args": ["--no-sandbox", "--disable-setuid-sandbox"]
   }
   ```
   - Required when running as root
   - Security consideration: Only on trusted servers

**Storage State Persistence:**
```python
context.storage_state(path="~/.config/browser/storage_state.json")
```
- Saves cookies, localStorage, sessionStorage
- Survives browser restarts
- JSON format for easy inspection
- Store securely (contains auth tokens!)

**Debug Mode:**
```python
if DEBUG:
    page.screenshot(path="/tmp/debug_screenshot.png")
```
- Always capture screenshots on failure
- Timestamp for debugging
- Check `/tmp/` for evidence
