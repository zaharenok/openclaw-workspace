# GitHub MCP - Полная документация

## 📋 Доступные инструменты: 37

### Работа с репозиториями
- `create_repository` - создать репозиторий
- `search_repositories` - поиск репозиториев
- `fork_repository` - форк репозитория

### Ветки и коммиты
- `create_branch` - создать ветку
- `list_branches` - список веток
- `list_commits` - история коммитов
- `get_commit` - информация о коммите
- `push_files` - пуш изменений

### Файлы
- `get_file_contents` - прочитать файл
- `create_or_update_file` - создать/обновить файл
- `delete_file` - удалить файл

### Issues
- `list_issues` - список issues
- `search_issues` - поиск issues
- `issue_read` - прочитать issue
- `issue_write` - создать/обновить issue
- `add_issue_comment` - комментарий к issue
- `sub_issue_write` - sub-issues

### Pull Requests
- `create_pull_request` - создать PR
- `list_pull_requests` - список PR
- `search_pull_requests` - поиск PR
- `pull_request_read` - прочитать PR
- `update_pull_request` - обновить PR
- `update_pull_request_branch` - обновить ветку PR
- `merge_pull_request` - мердж PR
- `pull_request_review_write` - рецензия PR
- `add_comment_to_pending_review` - комментарии к рецензии

### Code Search
- `search_code` - поиск кода
- `search_users` - поиск пользователей

### Releases & Tags
- `list_releases` - список релизов
- `get_latest_release` - последний релиз
- `get_release_by_tag` - релиз по тегу
- `list_tags` - список тегов
- `get_tag` - информация о теге

### GitHub Copilot
- `assign_copilot_to_issue` - назначить Copilot на issue
- `request_copilot_review` - запросить рецензию Copilot

### Прочее
- `get_me` - информация о текущем пользователе
- `get_label` - информация о лейбле
