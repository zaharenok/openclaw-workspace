# README.md - Автоматическое обновление ✅

## Что сделано

1. ✅ **Создан README.md** с полным описанием проекта
2. ✅ **Скрипт бэкапа обновлён** - теперь автоматически обновляет раздел "Что нового"
3. ✅ **Авто-обновление changelog** - при каждом бэкапе добавляется запись с timestamp

## Как это работает

### bin/backup-to-github.sh

При каждом запуске скрипт:
1. Проверяет какие файлы изменены
2. Формирует описание изменений
3. **Автоматически обновляет README.md** - раздел "Последние изменения"
4. Коммитит изменения
5. Пушит на GitHub

### Раздел "Что нового" в README.md

Обновляется автоматически:
```
### Последние изменения

**2026-01-30 21:29:34**
- ✅ Автоматический бэкап (2 файлов изменено)
- 📝 Изменённые файлы:
  - README.md
  - bin/backup-to-github.sh
```

## Примеры использования

### Ручной бэкап с обновлением README
```bash
./bin/backup-to-github.sh
```

**Результат:**
- README.md обновлён автоматически
- Все изменения закоммичены
- Всё запушено на GitHub

### Автоматический бэкап (каждые 6 часов)
- Cron job: "GitHub Backup - OpenClaw Soul"
- README.md обновляется сам
- Ничего делать не нужно!
