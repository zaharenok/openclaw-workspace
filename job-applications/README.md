# Job Applications

This directory contains job applications, cover letters, and related materials for various positions.

## Structure

```
job-applications/
├── README.md (this file)
└── [company-name]/
    ├── application-email.md
    ├── cv.md
    ├── research/
    └── follow-up/
```

## Application Process

1. **Research** - Gather company info, role requirements, fit analysis
2. **Draft Application** - Create tailored cover letter + email
3. **Review** - Check against requirements, proofread
4. **Submit** - Send application
5. **Track** - Update status, follow-up actions
6. **Archive** - Move to completed/accepted/rejected

## Status Labels

- `draft` - In progress
- `ready` - Ready to send
- `submitted` - Sent, waiting response
- `interviewing` - Interview scheduled/in progress
- `offer` - Received offer
- `accepted` - Accepted position
- `rejected` - Rejected
- `withdrawn` - Withrew application

## Templates

Each application folder should include:
- `application-email.md` - Cover letter + email text
- `cv.md` - Tailored CV/resume (optional)
- `research/` - Company research, role analysis
- `follow-up/` - Follow-up emails, thank you notes
